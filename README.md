Proyektin demo versiyası:
https://crud-app-murex-chi.vercel.app/

Bu proyekt sadə bir istifadəçi idarə etmə sistemidir
Frontend tərəfində React Redux toolkit və Tailwind Css istifadə etmişəm.Form validation üçün formik və Yup istifadə olunub. 
Backend üçün isə ExpressJS və MongoDB istifadə edərək POST,GET,DELETE VE PUT metodlarını icra edə biləcəyimiz kicik bir APİ yazmişam

Proyektdə isitfadəçi əlavə etmək, silmək, güncəlləmək və axtarış etmək funksiyaları mövcuddur.

Proyekti demo versiyasında daha ətraflı nəzərdən keçirə bilərsiniz.
Backend pulsuz serverə deploy edildiyi üçün demo versiyaya baxarkən serverdən cavab gec gələ bilər
